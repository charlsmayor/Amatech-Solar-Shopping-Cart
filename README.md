# Amatech Solar Shopping Cart

## Features
- User-friendly interface for shopping solar products
- Secure payment processing
- Order tracking and management
- Product reviews and ratings

## Installation Instructions
1. Clone the repository: `git clone https://github.com/charlsmayor/Amatech-Solar-Shopping-Cart.git`
2. Navigate to the project directory: `cd Amatech-Solar-Shopping-Cart`
3. Install the required dependencies: `npm install`
4. Start the application: `npm start`

## Usage Guide
- Navigate to the homepage to view available products.
- Add items to your cart by clicking the "Add to Cart" button.
- Proceed to checkout to finalize your purchase.

## System Requirements
- Node.js v12 or higher
- NPM v6 or higher
- A modern web browser (Chrome, Firefox, etc.)

## Troubleshooting
- If you encounter issues during installation, ensure that Node.js and NPM are properly installed on your system.
- For runtime errors, check the console for detailed error messages.
